## Module Structure

The module is divided into **4 Units** across 10 teaching weeks:

| Week | Unit | Lecture Topic | Lab / Tutorial |
|---|---|---|---|
| 1 | Unit 1 | Introductory Content - What is AI? | Getting Started with Google Colab / Python Refresher |
| 2 | Unit 1 | Modern AI Applications, Technologies and Ethics | AI Ethics Discussion |
| 3 | Unit 1 | Mathematics for AI and Data Science | Mathematics and Logic Refresher |
| 4 | Unit 2 | Introduction to Data Science | Mathematical Foundations in Excel and Python |
| 5 | Unit 2 | Data Science Concepts | Manipulating and Visualising Data in Excel and Python |
| 6 | Unit 3 | Solving Problems by Search I | Assessment Week (Unit 1 & 2) |
| 7 | Unit 3 | Solving Problems by Search II | Solving Problems by Searching in Python |
| 8 | Unit 3 | Constraints Satisfaction and Shortest Path | Graph Colouring and Shortest Path in Python |
| 9 | Unit 4 | Introduction to Machine Learning | Teachable Machines |
| 10 | Unit 4 | Introduction to Artificial Neural Networks | Machine Learning in Python |
| 11 | - | AI and Robotics | Assessment Week (Unit 3 & 4) |
| 12 | - | - | Assessment Catch-up / Re-sits |

<a id="labs-in-this-repo"></a>
## Labs in This Repo

### Unit 1 - Foundational and Mathematical Concepts in AI and Data Science
Covers the history and philosophy of AI, key definitions and approaches, AI ethics, and the mathematical foundations needed for AI and data science.

- [x] Week 1 - Python I: Basic Python (data types, strings, lists, dictionaries, tuples, functions, classes)
- [ ] Week 1 - Python II: NumPy library exercises (arrays, statistics, matrix operations)
- [ ] Week 2 - AI Ethics tutorial
- [ ] Week 3 - Mathematics and Logic Refresher (vectors, matrices, probability, logic)

### Unit 2 - Data Science Fundamentals
Covers the data science pipeline, statistical concepts, data cleaning, manipulation and visualisation using Python and Excel.

- [ ] Week 4 - Mathematical Foundations in Excel and Python
- [ ] Week 5 - Manipulating and Visualising Data in Excel and Python (Pandas, Matplotlib)

### Unit 3 - Algorithmic Problem Solving
Covers problem formulation, state spaces, uninformed and informed search algorithms, constraint satisfaction problems and shortest path algorithms.

- [ ] Week 7 - Uninformed Search in Python (BFS, DFS)
- [ ] Week 8 - Graph Colouring and Shortest Path in Python (A*, constraint satisfaction)

### Unit 4 - Machine Learning and Artificial Neural Networks
Covers supervised and unsupervised learning, classification, regression, neural network architecture, training and evaluation.

- [ ] Week 9 - Teachable Machines (Google)
- [ ] Week 10 - Machine Learning in Python (scikit-learn, neural networks)
